      Virtual Pet

      Virtual Pet is a java application that includes a game loop. The user is able to feed, water and walk a pet. The actions the user does to take care of the pet will cause either a positive or negative reaction to the pet's needs. This application was built using basic java fundamentals and test driven development.

      Virtual Pet Shelter

      In Virtual Pet Shelter the user acts as a pet shelter volunteer and has the ability to take care of multiple pets or a single pet at a time. This application is built off Virtual Pet and has all of Virtual Pet's capabilities.

      Virtual Pet Amok

      Virtual Pet Amok is a java application that is an extension of Virtual Pet Shelter. In this application, the user acts as a pet shelter volunteer. Virtual Pet Amok is able to interact with different types of animals using object oriented programming plus has all the capabilities of Virtual Pet Shelter.
